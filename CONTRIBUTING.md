# Contributing

New to the repo? Start with **[docs/developer-onboarding.md](./docs/developer-onboarding.md)**
and the doc index at **[docs/README.md](./docs/README.md)**.

## Branch Naming

Feature:

feature/<feature-name>

Bug Fix:

fix/<bug-name>

Documentation:

docs/<doc-name>

---

## Commit Convention

feat:

new feature

fix:

bug fix

docs:

documentation

refactor:

code improvement

chore:

maintenance

---

## Pull Requests

1. Create feature branch
2. Open PR
3. Request review
4. Merge after approval
