# Contributing

## Branch Naming

Feature:

feature/<feature-name>

Bug Fix:

fix/<bug-name>

Documentation:

docs/<doc-name>

---

## Commit Convention

feat:

new feature

fix:

bug fix

docs:

documentation

refactor:

code improvement

chore:

maintenance

---

## Pull Requests

1. Create feature branch
2. Open PR
3. Request review
4. Merge after approval
