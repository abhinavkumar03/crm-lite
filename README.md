# CRM Lite

Production-style CRM Lite MVP built to demonstrate modern backend engineering practices.

## Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS

### Backend

- Go
- Gin

### Database

- PostgreSQL

### Cache

- Redis

### Infrastructure

- Docker
- Kubernetes
- Render

---

## Features

### Authentication

- Register
- Login
- JWT Authentication

### Leads

- Create Lead
- Update Lead
- Delete Lead
- Track Status

### Contacts

- Create Contact
- Update Contact
- Delete Contact

### Tasks

- Create Task
- Assign Task
- Track Progress

### Analytics

- Lead Metrics
- Task Metrics
- Dashboard Overview
